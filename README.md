# NBA EDA Dashboard

A simple Streamlit app for exploratory data analysis (EDA) on NBA data — visualizing trends, stats, and patterns through interactive charts.

## Features

- Interactive data exploration of NBA stats
- Visualizations built with Matplotlib, Seaborn
- Filters/widgets to slice the data (e.g. by team or player)

## Tech Stack

- **Python**
- **Streamlit** – app framework
- **Pandas** – data manipulation
- **Matplotlib** – visualizations
- **Seaborn** - vizualizations
- **lxml**


## Getting Started

### Prerequisites

- Python 3.8+

### Installation

1. Clone the repository

   ```bash
   git clone https://github.com/dragangolic/NBA-EDA.git
   cd your-repo-name
   ```

2. Create a virtual environment

   ```bash
   python -m venv venv
   source venv/bin/activate  
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

### Running the App

```bash
streamlit run main.py
```

The app will open in your browser at `http://localhost:8501`.
